# 🚁 AI Drone Vision System

A real-time AI-powered drone surveillance system built using **YOLOv8, OpenCV, and Flask**.

---

## 🔥 Features

- 🎥 Live video streaming  
- 🧠 Real-time object detection (YOLOv8)  
- 📊 Interactive detection dashboard  
- 🗺️ GPS & telemetry simulation  
- 📷 Snapshot capture  
- 🎮 Camera ON/OFF control  
- ⚡ Optimized performance (frame skipping + compression)  

---

## 🛠️ Tech Stack

- Python  
- OpenCV  
- Flask  
- Ultralytics YOLOv8  
- HTML / CSS / JavaScript  

---

## 🚀 How to Run

```bash
git clone https://github.com/kanakk020/drone-detection.git
cd drone-detection
pip install -r requirements.txt
python app.py

Then open in browser:

http://localhost:5000
